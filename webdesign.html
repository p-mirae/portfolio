WEB SITE

概要
建設会社のWebサイト。
制作目的
木村工業株式会社様は父が務める会社で、サイトリニューアルが長らく止まっていると知り、自ら社長にお願いして、デザインのみではありますが担当させていただきました。
デザイン

お使いのサイトは10年ほど前に制作されたもので、公開後一度も更新されていなかったため、施工事例や実績が古くなっていること、業界的に見る人は限られているが、業界を知らない人が見ても分かりやすいサイトにしたいというご要望をいただきました。

ヒアリングを通して、最も伝えたいこととして「施工事例」「既存顧客、新規顧客へのアピール」「きつい、汚い、危険（3K）と嫌がられる業界である建設業が輝かしく素晴らしい仕事である事」という点が挙げられました。そのため、全ページの下部に「施工事例」のスライダー、「採用情報」に飛ぶCTAボタンを設置。また、すぐ問い合わせができるようにヘッダーとフッターに問い合わせボタンを設置しました。

サイトに期待する効果としては、「興味や魅力を感じてもらい、応募に繋げる」「新規顧客への信頼度を向上させる」ことを目的にデザインしました。

「建設業をもっとカッコよく、もっと身近に」をコンセプトに、特に若年層に響く「カッコよさ」を重視し、活気を感じさせるデザインを意識しました。濃いめのイエローを基調に、太く力強いフォントを使用することで、エネルギッシュでインパクトのある印象を演出しています。

今回はデザインのみ担当し、実装については制作会社と進行中です。

※ポートフォリオに掲載しているサイトは、デザイン確認用としてコーディングしたものになります。

制作時間
2ヶ月

WEB SITE

概要
建設会社のWebサイト。
制作目的
施工実績を最新の情報に更新し、建設業の魅力をより伝えるためのリニューアル提案として制作したデザインカンプの一案です。（※内容は架空のものです）
デザイン

このデザインは、建設業の「3K（きつい・汚い・危険）」というイメージを払拭したいというご要望を受け、白を基調としたクリーンで洗練された印象に仕上げました。

どの世代の方にも直感的に使いやすく、視認性の高い設計を追求し、誰もが迷うことなく必要な情報にたどり着けるよう工夫しました。全体の色使いはモノトーンを基調とし、装飾を最小限に抑えることでシンプルさを保ちつつ、細めのフォントを採用することで上品で洗練された印象を演出。

無駄を削ぎ落としながらも機能性を確保し、流行に左右されることなく長く使い続けられるデザインにしました。

制作時間
2週間

WEB SITE

概要
ポートフォリオ
制作目的
就職活動の際に履歴書ではアピールできない、「自分」を表現するために制作。
デザイン

今までの制作作品や経歴を一冊のノートに記録するような感覚でまとめたく、ノート風のグリッド背景を取り入れました。

ノートに記録をまとめていく様子を表現するため、メインビジュアルの文字が浮かび上がるアニメーションや、手書き風のフォント、線が描かれていくアニメーションを取り入れることで、視覚的な楽しさとストーリー性を感じられるデザインにしました。

落ち着いた性格だけども、明るく活動的な部分もあることを黄色を使って表現し、モノトーン配色+アクセントカラーにすることでメリハリを出しています。

制作時間
3週間

WEB SITE

概要
まちづくりに関するwebサイト
制作目的
実際にクライアント様をお呼びし、チームで制作したうちの案のひとつです。（プライバシー保護のため地域名・テキストは架空の内容に差し替えています）
デザイン

まちづくりの活動を住民の皆様に知ってもらいたいというご要望を受け、ヒヤリングから「スマホ閲覧が中心」というターゲット層の特徴を反映した、モバイル特化型のデザインにしました。

閲覧者が「自分事」として捉えられるよう、共感を喚起する訴求メッセージをコンテンツ間に組み込むことで、情報を一方的なものにせず主体的に捉えられるような仕組みにしました。

また、「幅広い世代に響く」というコンセプトを基盤に、「白（新しい始まり・誠実さ）」、「緑（自然の調和・安心感）」、「オレンジ（親しみやすさ・温かみ）」を使用し、視覚的な親しみと印象に残る配色にしました。

制作時間
1週間
